# Portfolio

Personal portfolio website of **Mehul Pradipkumar Balsara** — Machine Learning Engineer & Data Scientist.

🔗 **Live site:** https://portfolio-sandy-one-78.vercel.app/

## Overview

A one-page, dark-themed portfolio built with React and Framer Motion. Features an interactive canvas-based neural network in the hero, scroll-triggered animations, and real project data pulled from my GitHub and Kaggle work.

## Tech Stack

- React 18 + Vite
- Framer Motion (animations)
- Vanilla Canvas API (neural network hero background)
- Plain CSS (no framework)

## Sections

- **Hero** — intro, headline, quick stats
- **About** — background and approach
- **Skills** — core toolkit with proficiency indicators
- **Projects** — 7 shipped projects (forecasting, classification, EDA, and AI agent systems)
- **GitHub** — profile summary
- **Experience** — current trajectory and internship search
- **Contact** — email and social links

## Run locally

```bash
git clone https://github.com/Mehul5124/Portfolio.git
cd Portfolio
npm install
npm run dev
```

Open the printed local URL (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
```

Outputs a static site to `dist/`.

## Deploy

Deployed on [Vercel](https://vercel.com) — framework preset **Vite**, auto-detected. Every push to `main` triggers a redeploy.
